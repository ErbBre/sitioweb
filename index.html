Acerca de mí

Durante mi estadía en la carrera de Ing. software con inteligencia artificial aprendí muchos conceptos sobre el área informático
especifico el proceso de desarrollo o ciclo de vida del software, desde la definición de necesidades, maquetado hasta el mantenimiento
del mismo, además como parte del proceso se tiene en cuenta conceptos como UX(User Experience) y UI(User interfaz) básicamente
con esto hacemos que el software sea intuitivo y facil de operar por el usuario además de tener un aspecto llamativo, este proceso
es fundamental en el desarrollo. Como parte del desarrollo tuve la oportunidad de analizar bases de datos, datos que por lo general
lo generan los usuarios finales y nos sirven para realizar procesos de estadística.

Asimismo, como parte de mis primeros pasos, en cooperación grupal realice un proyecto de desarrollo
de un sistema de inventario en el que aplicamos nuestros conocimientos hasta tener el producto final en
funcionamiento. Pudimos vivir la experiencia de llevar a cabo las tareas en fechas establecidas para simular
un caso real.

Durante el día suelo practicar nuevas habilidades y reforzar los que ya tengo ya que en el área tecnológico constantemente llegan
nuevas actualizaciones y es de vital importancia saber de ello.

Y si hablamos de código, hace poco desarrollé un sistema de registro de pagos completamente en Python el cual tiene como
principal función llevar el registro de cada cliente y emitir reportes en Excel, para almacenar los datos usé SQLite como base
de datos ya que es portable y no requiere de muchas configuraciones, además de ser el más adecuado para el requerimiento
del usuario.

Portafolio

Calculadora de resistencia

Aplicación de escritorio con una interfaz de botones y widgets seleccionables desarrollado en Python. Esta App facilita el cálculo del valor de la resistencia (componente electrónico), se usa seleccionando la cantidad de bandas que tiene y seleccionando el mismo en la interfaz consta de opciones para resistencias de 3,4,5 y 6 bandas y el valor nominal es medido en Ohmios así como también el margen de variación.

Chat en área Local

Esta App permite el envío y recepción de mensajes a través de la conexión TCP/IP uno de ellos cumple la función de servidor y el segundo el cliente, la conexión se da siempre y cuando ambos dispositivos se encuentren en la misma red local la aplicación cliente hace la petición de conexión a través de un IP y puerto.

Sistema de registro

Implemente esta App en el área de servició técnico donde tiene las características de registrar un equipo con sus datos correspondientes junto a los datos del cliente, en la pestaña de inicio se muestran cantidad de equipos y el estado en el que se encuentran en su proceso de reparación, puede generar un ticket de recepción e imprimir en una impresora térmica de 80mm los datos que la App genera se almacenan en SQLite que es un gestor de base de datos ligero y portable, el diseño de la ventana en adaptable a casi cualquier tamaño ya que se tuvo en cuenta el diseño responsive al momento de la planificación.

Interfaz Login

Un formulario con dos campos para datos de inicio de sesión usando HTML con estilos CSS responsive que se adapta al tamaño de la pantalla del dispositivo en el que se pueda mostrar.

Descargar Playlist

Desarrollado en Python usando el módulo Pytube, tiene una interfaz sencilla donde al pegar el enlace del PlayList de YouTube y presionar el botón de ´Descargar´, automáticamente se procede con la descarga de todos los videos que conforma la PlayList en la misma carpeta donde se encuentra el ejecutable y en la parte inferior se muestra el resultado de las descargas donde también se informa del posible error que pueda ocurrir en cada descarga.

Sistema de cobranza

Con esta implementación se busco facilitar el registro de pagos de cada cliente ya que anteriormente el proceso era escrito, muy tedioso y se ocupa más tiempo, así que este programa tiene como principal función almacenar los datos de pagos, cuenta con filtro de búsqueda donde a cada letra que se ingresa se van mostrando las coincidencias, así como también tiene la posibilidad de exportar los datos en archivos Excel.
